## Актуальные числа 2026 (с основанием)

- **Фиксированные взносы ИП = 57 390 ₽** (ст. 430 НК РФ, индексация на 2026). Срок — 28.12.2026.
- **1% с дохода свыше 300 000 ₽**, потолок доп. взноса за 2026 = **321 818 ₽**. Срок 1% за 2025 — **01.07.2026**, за 2026 — 01.07.2027.
- **МРОТ 2026 = 27 093 ₽** (ФЗ № 429-ФЗ). **Предельная база взносов = 2 979 000 ₽**.
- **Лимит дохода УСН 2026 ≈ 490,5 млн ₽** (450 млн × дефлятор 1,090). Численность ≤ 130, ОС ≤ 218 млн.
- **НДС на УСН (с 2025):** освобождение до **60 млн ₽**; **5%** при 60–250 млн; **7%** свыше 250 млн. (Плановое снижение порога: 2027 → 15 млн, 2028 → 10 млн — проверять на nalog.gov.ru.)
- **Повышенные ставки УСН 8%/20% отменены с 2025** — переходного коридора нет.
- **Перечень расходов УСН «Д−Р» с 01.01.2026 — открытый** (любые обоснованные по ст. 252 НК).
- **Декларация УСН** — новая форма по приказу ФНС **ЕД-7-3/1017@ от 26.11.2025**, действует с 28.02.2026 (за 2025 год — уже по ней). Новое: строки 150–162 («Доходы») и 290–320 («Д−Р») — взносы ИП нарастающим итогом по кварталам.
- **КЭП с 01.04.2026** — новые ГОСТ 34.12-2018 / 34.13-2018; нужен свежий КриптоПро.

## Реализовано (фаза v7, 2026-06-17)

Многоагентный аудит + доводка до паритета с Эльбой:

- Реквизиты ИП с ОКТМО и выбором ставки НДС
- Авто-стаж из даты приёма (`lib/stazh.ts`) + эффективный стаж = авто или ручной
- Прогрессивный НДФЛ 13/15/18/20/22% нарастающим итогом (ст. 224 НК)
- Аванс-НДФЛ по двум периодам уведомлений (с 2023): аванс 1–22-го / расчёт 23-го–конец
- Заработок по годам → базы отпускных (÷12÷29,3) и больничных (÷730)
- Калькуляторы по сотруднику: зарплата, отпускные, больничные, алименты
- Сводка по штату: НДФЛ, взносы, чистые выплаты
- Карточка сотрудника с печатью + увольнение
- НДС-2026 (ФЗ № 425-ФЗ): общая ставка 22%, порог освобождения 20 млн, спец-ставки 5%/7% — сквозной расчёт
- Книга продаж + декларация НДС (XML-выгрузка)
- Документы: исходящие (счёт/акт/накладная/УПД) + входящие
- Налоговый календарь: срочные / будущие задачи + архив
- Раздел «Налоговая»: ЕНС сальдо, сверка, письма + FnsGateway (ст. 102 НК — ручной ввод + имитация)
- Уведомления КНД 1110355, раздел «Полезные документы»
- Снимки / откат / бэкап / журнал действий (`lib/storage/storeAdmin.ts`), экран Admin
- ТОРГ-12 / УПД в документах
- P&L (отчёт о прибылях и убытках) в разделе «Деньги»
- Проверка контрагента по ИНН (DaData + демо-словарь)
- Шаблоны договоров (ContractDoc)
- Мобильная адаптация (выезжающее меню)
- Сумма прописью (`lib/numberToWords.ts`) в документах
- Статусы счетов + мини-дебиторка
- Автозаполнение контрагента по ИНН
- Decimal-агрегация в `compute.ts` (устранён float-дрейф)
- Гард НДС > 450 млн (УСН утрачена → ОСНО)
- Тесты: Python 73 + TS 160
