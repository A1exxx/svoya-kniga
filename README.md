# СвояКнига (рабочее имя)

Своя онлайн-бухгалтерия для **ИП на УСН** (Россия) — аналог Контур.Эльбы для собственного использования с прицелом на продукт.

> Полный план проекта: `C:\Users\user\.claude\plans\functional-crafting-blanket.md`

## Принципы
- Узкий точный домен: ИП на УСН (не «ещё один ERP», без двойной записи).
- **Сначала функционал, который считает правильно**, потом интерфейс, потом коннекторы/отправка.
- Все ставки/лимиты/взносы — **параметры по годам** (меняются ежегодно), правятся без кода.
- Корректность сверяет **бухгалтер**. Приложение — инструмент; ответственность на ИП/бухгалтере.

## Этап 1 (текущий)
Ядро калькуляторов + формирование отчётности + удобный интерфейс.

```
backend/
  taxcore/         # ядро расчётов (наша IP)
    params.py      # параметры налогов по годам (ставки, взносы, лимиты, сроки)
    models.py      # типы данных (Decimal-деньги)
    vznosy.py      # страховые взносы ИП (фикс + 1% свыше 300т)
    usn.py         # УСН: доходы 6% / доходы-расходы 15%, авансы, мин. налог
    ens.py         # ЕНС: авансы по срокам, уведомления, налоговый календарь
  tests/           # pytest — канонические сценарии расчётов
```
